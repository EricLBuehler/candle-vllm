系统要求

组件 要求
操作系统Linux (CUDA) 或 macOS (Metal)
Rust 工具链Rust 1.75+（仅源码编译需要）
CUDA11.x、12.x 或 13.0（Linux GPU）
Python3.8+（仅量化转换脚本需要）

硬件要求

平台 最低 GPU 推荐特性
CUDA (Linux) SM70+ (V100) 8GB+ SM90 (H100) 用于 FP8/CUTLASS
Metal (macOS) Apple Silicon M1+，8GB+ M4 Pro/Max 推荐

一键安装

最快的方式——自动检测平台并安装预编译二进制或 DEB 包：

$ curl -sSL https://ericlbuehler.github.io/candle-vllm/install.sh | bash

源码编译

# 克隆仓库
git clone git@github.com:EricLBuehler/candle-vllm.git
cd candle-vllm

# 编译（CUDA 11+/12+/13.0）
# 对于 SM70/SM75，移除 flashinfer,cutlass 特性
cargo install --features cuda,nccl,flashinfer,cutlass --path .
# 克隆仓库
git clone git@github.com:EricLBuehler/candle-vllm.git
cd candle-vllm

# 编译（Metal / Apple Silicon）
cargo install --features metal --path .
💡 提示

SM70/SM75 GPU（V100、T4）请移除 flashinfer 和 cutlass 特性。SM80+（A100）支持全部特性。SM90+（H100）额外支持 CUTLASS FP8 算子。

Docker 部署

# 默认编译（自动检测 SM 版本）
./build_docker.sh "cuda,nccl,flashinfer,cutlass"

# 指定 SM 版本和 CUDA 版本
./build_docker.sh "cuda,nccl,flashinfer,cutlass" sm_90 13.0.0

启动模型

安装完成后，使用 HuggingFace 模型 ID 或本地路径即可启动推理服务：

  1. 使用 HuggingFace 模型

    # FP8 模型 + Web UI
candle-vllm --m Qwen/Qwen3.6-27B-FP8 --ui-server

# GGUF 量化模型
candle-vllm --m unsloth/Qwen3.5-122B-A10B-GGUF --f Q3_K_S --d 0,1 --ui-server

  2. 使用本地模型

    # 本地 Safetensors 目录
candle-vllm --d 0,1 --m /home/Qwen3-30B-A3B-Instruct-2507/ --ui-server

# 本地 GGUF 文件
candle-vllm --m /home/data/model-Q4_K_M.gguf --ui-server

测试 API

服务启动后，默认监听 http://localhost:2000，使用标准 OpenAI 格式调用：

curl http://localhost:2000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
  "model": "qwen",
  "messages": [{"role": "user", "content": "Hello!"}],
  "max_tokens": 128,
  "stream": true
}'

Web UI

添加 --ui-server 标志即可启动内置 ChatGPT 风格 Web UI。UI 端口 = API 端口 - 1（例如 API 在 2000，UI 在 1999）。
