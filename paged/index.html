核心特性

为生产环境打造

从单张 GPU 到多节点集群，从 FP16 到 2-bit 量化，Candle-vLLM 覆盖全场景推理需求。

⚡

极致性能

原生 Flash Attention、FlashInfer 后端、CUDA Graph 捕获，解码速度最高达 251 tks/s。

🗜️

TurboQuant KV 压缩

基于 Walsh-Hadamard 变换的 KV 缓存压缩（2-4 bit），最高 4.7× 压缩比，几乎无质量损失。

🌍

跨平台

同一代码库支持 CUDA（Linux）和 Metal（macOS），相同 API，开箱即用。

🏭

生产就绪

OpenAI 兼容 API、内置 ChatGPT 风格 Web UI、MCP 工具调用、流式输出。

📦

轻松部署

一行安装脚本、Docker 镜像、或从源码构建。支持 DEB 包和预编译二进制。

🔧

高度可扩展

基于 Trait 的架构设计，快速实现新模型管线。支持 Safetensors、GGUF、GPTQ/Marlin、FP8/FP4。

🖥️

多 GPU 与多节点

多进程/多线程张量并行，基于 TCP 的多节点推理，无需 MPI。

🧠

连续批处理

vLLM 风格的调度器，支持分块预填充、前缀缓存、抢占式调度和 KV 块管理。

🔌

MCP 与工具调用

内置 Model Context Protocol 支持，兼容 OpenAI 工具调用格式，多种模型解析后端。

模型支持

广泛的模型覆盖

支持主流 LLM 模型族，包括 Safetensors、GGUF、GPTQ/Marlin、FP8/FP4 多种格式。
