绑定示例

# TCP（指定端口）
candle-vllm --h 127.0.0.1 --p 8000 --m Qwen/Qwen3.6-27B-FP8

# TCP（地址含端口）
candle-vllm --h 127.0.0.1:8000 --m Qwen/Qwen3.6-27B-FP8

# IPv6
candle-vllm --h '[::1]:8000' --m Qwen/Qwen3.6-27B-FP8

# Unix Socket
candle-vllm --h unix:///tmp/candle-vllm.sock --m Qwen/Qwen3.6-27B-FP8

GET /v1/models — 模型列表

列出当前加载的模型及其能力和最大序列长度。

curl http://localhost:2000/v1/models

POST /v1/chat/completions — 聊天补全

完全兼容 OpenAI Chat Completion API 格式，支持流式输出、工具调用和多模态输入。

curl http://localhost:2000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
  "model": "qwen",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Explain PagedAttention."}
  ],
  "max_tokens": 256,
  "temperature": 0.7
}'
curl http://localhost:2000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
  "model": "qwen",
  "messages": [{"role": "user", "content": "Hello!"}],
  "stream": true
}'
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:2000/v1",
    api_key="not-needed"
)

response = client.chat.completions.create(
    model="qwen",
    messages=[{"role": "user", "content": "Hello!"}],
    stream=True
)

for chunk in response:
    print(chunk.choices[0].delta.content, end="")

POST /v1/embeddings — 嵌入

curl http://localhost:2000/v1/embeddings \
  -H "Content-Type: application/json" \
  -d '{
  "model": "qwen",
  "input": "Candle-vLLM is a Rust inference engine",
  "embedding_type": "mean"
}'
字段描述
embedding_type池化模式：mean（默认）或 last
encoding_format返回格式：float 或 base64

工具调用

支持 OpenAI 格式的工具调用，通过 tools 字段定义工具：

curl http://localhost:2000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
  "model": "qwen",
  "messages": [{"role": "user", "content": "What is the weather in Tokyo?"}],
  "tools": [{
    "type": "function",
    "function": {
      "name": "get_weather",
      "description": "Get weather for a city",
      "parameters": {
        "type": "object",
        "properties": {
          "city": {"type": "string", "description": "City name"}
        },
        "required": ["city"]
      }
    }
  }]
}'

流式输出

设置 stream: true 即可启用 Server-Sent Events (SSE) 流式输出。响应中包含 usage 信息，含 cached_tokens 字段反映前缀缓存命中。
